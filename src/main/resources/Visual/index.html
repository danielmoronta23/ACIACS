Anti-CoVID19 Intelligent Access Control System
Mas Información

Servicios

Objectivos

Implementar un mecanismo de luces, sensores y actuadores, que permitan controlar el flujo de personas que entrean o salen de un local.

Desarrollar un sistema de reconocimiento facial capaz de identificar si un individuo está utilizando o no, una mascarilla de protección.

Diseñar y desarrollar un mecanismo autónomo, que sea capaz de realizar el control y chequeo de acceso a personas en un establecimiento de insumos, con el fin de poder cumplir con las medidas de bioseguridad para la prevención de la enfermedad del CoVID19.

Construir un sistema de medición de temperatura corporal a distancia, capaz de identificar si un individuo posee o no, alta temperatura.

Desarrollar una aplicación web alojada en la nube, que permita la configuración y/o supervisión de los sistemas y mecanismo de control y chequeo.

Sobre Nosotros

Resumen Ejecutivo

Este proyecto ha desarrollado un sistema de control de acceso inteligente para clientes, enfocado en supermercados y tiendas, para la protección y prevención del Covid-19. Mediante la utilización de inteligencia artificial, sensores y Raspberry Pi para evaluar si cada una de las personas que desean acceder a estas edificaciones, poseen mascarillas, temperatura corporal normal y manteniendo la cantidad de personas permitidas dentro del local.

El sistema posee una aplicación web, para la rápida y fácil gestión de este. Además, ha sido implementada en un ambiente en la nube, de forma que el acceso a esta sea rápido y eficiente desde cualquier ubicación. Dicha aplicación presenta un entorno amigable para el usuario y mantiene una llana área de configuración, de modo que su utilización sea asequible a cualquier tipo de usuario.

Este esfuerzo de desarrollo ha sido para eficientizar el control de acceso que existe hoy en día, en los supermercados y tiendas del mundo. Debido a la necesarias y obligatorias medidas de prevención del Covid-19 que tienen que implementarse en estos sectores comerciales. El Control de Acceso Inteligente Anti-Covid19 (ACIAC) brinda una solución automatizada, moderna y más eficiente que las metodologías de control tradicionales. Lo cual permite un mejor flujo de entrada y salida de clientes, como también brinda una mejor imagen de modernización y calidad a la empresa.
